# Camera Calibration Studio

Clean frontend bundle containing the work completed so far:

- React + TypeScript + Vite scaffold
- Calibration data contracts
- Realistic mock calibration project and frames
- Responsive dark glassmorphism design system
- Landing page and workspace navigation
- Extracted landing hero
- Interactive React Three Fiber calibration scene

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The workspace panels still use placeholders. Upload, dataset review, calibration execution, results and export components are the next implementation stages.
